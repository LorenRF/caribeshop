# CaribeShop 🛍️

Plataforma web de comercio electrónico desarrollada como práctica técnica para vacante de desarrolladora. Utiliza ASP.NET Core para el backend y Angular para el frontend.

## Tecnologías

- **Frontend**: Angular 15+, Bootstrap 5, RxJS, LocalStorage
- **Backend**: ASP.NET Core Web API, JWT, Entity Framework Core, SQL Server

## Estructura del Proyecto

caribeshop/
├── frontend/ --> Angular App
├── backend/ --> ASP.NET Core Web API


---

## 🚀 Cómo ejecutar el proyecto

### Backend

```bash
cd backend
dotnet restore
dotnet run

Se ejecuta en https://localhost:5001 (configurable)



Frontend

cd frontend
npm install
ng serve

Se ejecuta en http://localhost:4200

📦 Funcionalidades destacadas
Autenticación JWT segura con hashing SHA256

Gestión completa de carrito de compras

Diseño responsive, accesible y coherente visualmente

División limpia de componentes, servicios y rutas protegidas
